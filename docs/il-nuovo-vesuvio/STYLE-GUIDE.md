## Über das Restaurant

Das Restaurant Il Nuovo Vesuvio besteht seit September 1997 unter der Leitung des mehrfach ausgezeichneten Chefkoch Vito Lamanna. Geboten wird eine bodenständige und kreative, original italienische Küche mit Weinen aus allen Regionen Italiens.

Das gemütlich eingerichtete Restaurant bietet mit dem angeschlossenen Nebenraum 95 Sitzplätze. Im Sommer genießen die Gäste ihren Aufenthalt auf der stilvoll gestalteten Gartenterrasse im Grünen mit weiteren 90 Sitzplätzen.

## Kreative Design-Empfehlungen

### 1. Empfohlenes Layout-Konzept

**"Elegante Trattoria"** - Ein warmes, einladendes Design das italienische Gastfreundschaft vermittelt:

- **Hero:** Fullscreen-Slider mit Restaurant-Atmosphäre + überlagerndem Logo
- **Über uns:** Split-Screen mit Bild links, Text rechts (Storytelling)
- **Speisekarte:** Elegante Cards mit Kategorien + Hover-Effekten
- **Galerie:** Masonry-Grid mit Lightbox
- **Kontakt:** Google Maps + Kontaktformular nebeneinander

### 2. Signature-Effekt

**Warme Gradient-Overlays + Weinrot-Akzente:**
- Sanfte Altrosa-zu-Weiß Gradienten auf Sektionen
- Weinrote Akzentlinien unter Überschriften
- Goldene/Bronze Dekorelemente für Premium-Feeling
- Subtile Textur-Overlays (leichtes Papier/Stoff-Muster)

### 3. Animations-Level

**Moderat** (Restaurant-typisch):
- Sanfte Fade-in Animationen beim Scrollen
- Elegante Hover-Effekte auf Bildern (leichter Zoom + Overlay)
- Smooth Scroll-Verhalten
- Dezente Parallax-Effekte auf Hero-Bildern
- Menü-Items mit stagger Animation

### 4. Besondere Sektionen

| Sektion | Beschreibung |
|---------|--------------|
| **Hero mit Video-Option** | Fullscreen-Slider + Video-Button für Restaurant-Atmosphäre |
| **"Il Nostro Chef"** | Chefkoch-Portrait mit Zitat und Geschichte |
| **Speisekarten-Preview** | 3-4 Signature-Gerichte als Appetizer |
| **Weinkeller-Sektion** | Dunkler Hintergrund, Wein-Highlights mit Regionen |
| **Terrassen-Galerie** | Horizontal-Scroll Galerie für Sommerterrasse |
| **Öffnungszeiten-Widget** | Prominente Darstellung mit "Jetzt geöffnet" Indikator |
| **Google Reviews** | Integration der 4.2 Sterne Bewertung |
| **Anfahrt** | Interaktive Karte + Wegbeschreibung |

### 5. Empfohlene Bildsprache

- **Warm & Einladend:** Warme Farbtemperatur, gedimmtes Licht
- **Authentisch:** Echte Gerichte, keine Stock-Fotos
- **Detail-Shots:** Nahaufnahmen von Speisen und Wein
- **Ambiente:** Restaurant-Interieur und Terrasse
- **Menschen:** Wenn möglich, Gäste beim Genießen

### 6. UI-Komponenten

| Komponente | Style |
|------------|-------|
| **Buttons** | Abgerundete Ecken (8px), Weinrot mit Hover zu Dunkelrot |
| **Cards** | Weicher Schatten, 12px Radius, Creme-Hintergrund |
| **Inputs** | Bordeaux-Border bei Focus, 4px Radius |
| **Navigation** | Transparent → Solid Altrosa bei Scroll |
| **Footer** | Dunkler (Dunkelbraun/Weinrot) mit hellem Text |
