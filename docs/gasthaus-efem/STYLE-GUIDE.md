## Hinweise zur Implementierung

### Testimonials-Sektion
Da keine Profilbilder von Reviewern verfügbar sind (Datenschutz), empfehle ich:
1. **Initialen-Avatare** mit Hintergrundfarbe aus Farbpalette
2. **Google Reviews Widget/Badge** als Alternative
3. **Zitat-Karten** ohne Personenfotos

### Beispiel Initialen-Avatar CSS
```css
.testimonial-avatar {
  width: 48px;
  height: 48px;
  border-radius: 50%;
  background: var(--primary-color);
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
  font-weight: 600;
  font-size: 1rem;
}
```
